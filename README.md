# Instalation

## [Oh-My-Zsh](https://github.com/robbyrussell/oh-my-zsh)

- `git clone git@github.com:GetAmbush/deepx-zsh-plugin.git ~/.oh-my-zsh/custom/plugins/deepx`
- add `deepx` to the list of plugins on your `.zshrc` file
